# Dossier Musique

Placez ici votre fichier audio de fond pour le projet.

## Instructions

1. Téléchargez ou créez votre piste audio préférée (format MP3 recommandé)
2. Nommez le fichier `background.mp3`
3. Placez-le dans ce dossier (`src/assets/music/`)

## À propos de "Khaled Talk"

Si vous souhaitez utiliser une musique spécifique comme "Khaled Talk", veuillez vous assurer d'avoir les droits nécessaires pour l'utiliser dans votre projet personnel. Cette musique pourrait être protégée par le droit d'auteur.

## Alternatives libres de droits

Vous pouvez trouver de la musique libre de droits sur des sites comme:
- Free Music Archive (freemusicarchive.org)
- YouTube Audio Library
- Bensound (bensound.com)
- Purple Planet Music (purple-planet.com)

Assurez-vous de vérifier les licences avant utilisation.
